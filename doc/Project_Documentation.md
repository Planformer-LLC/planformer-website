# Planformer Website Documentation

This folder is intended for documentation related to the Planformer website.

## Useful Commands

- **Clean, Run**: `rm -rf .next out && npm run dev` (Removes previous build output and starts the development server)
- **Build**: \`npm run build\`
- **Deploy**: \`npm run deploy\` (Cleans \`.next\` and \`out\`, builds the app, and deploys to Firebase Hosting)
